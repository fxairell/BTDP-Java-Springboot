di dalam folder lain.

1. Definisi
a) File dengan ekstensi java merupakan file berisi coding dengan bahasa pemrograman Java.
b) File dengan ekstensi xml merupakan file data dengan tag digunakan untuk menyimpan data.
c) File dengan ekstensi jsp merupakan file data dengan tag digunakan untuk menampilkan data pada
   halaman server java.

2. Daftar File
- Member.java
- HelloServlet.java
- LoginController.java
- web.xml
- index.jsp
- member.jsp

3. Deskripsi File
a) Member.java
	File Member.java merupakan file model untuk menyatakan mana variabel apa. Model digunakan agar
dapat terhubung dengan controller.
b) HelloServlet.java
	File HelloServlet.java merupakan file untuk menggunakan servlet untuk menampilkan pernyataan
sederhana.
c) LoginController.java
	File LoginController.java merupakan file controller menghubungkan halaman login sederhana dan
mendapatkan password agar dapat menuju halaman berikutnya.
d) web.xml
	File web.xml merupakan file untuk mendefinisikan pustaka mapping penggunaan servlet. File akan
menghubungkan penggunaan file dan ketentuan dari penggunaan servlet.
e) index.jsp
	File index.jsp merupakan file untuk menampilkan halaman login awal sederhana.
f) member.jsp
	File member.jsp merupakan file untuk menampilkan halaman sapa member sederhana setelah login.
