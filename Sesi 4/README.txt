3. Deskripsi File
a) Array.class
	File Array.class merupakan file hasil compile Array.java. File akan menjalankan fungsi yang sama
dengan file Array.java.
b) Array.java
	File Array.java merupakan file pengenalan penggunaan array dan bentuk-bentuk deklarasi array. File
disertai dengan beberapa contoh penggunaan array dan cara pemanggilan array secara sederhana.
c) Array2D.class
	File Array2D.class merupakan file hasil compile Array2D.java. File akan menjalankan fungsi yang
sama dengan file Array2D.java.
d) Array2D.java
	File Array2D.java merupakan file pengenalan penggunaan array multidimensi.
e) ContohArray.class
	File ContohArray.class merupakan file hasil compile ContohArray.java. File akan menjalankan fungsi
yang sama dengan file ContohArray.java.
f) ContohArray.java
	File ContohArray.java merupakan file pengenalan contoh array.
g) itprofesi.class
	File itprofesi.class merupakan file hasil compile itprofesi.java. File akan menjalankan fungsi
yang sama dengan file itprofesi.java.
h) itprofesi.java
	File itprofesi.java merupakan file latihan sederhana dalam menggunakan dan memanggil array.
i) node.class
	File node.class merupakan file hasil compile Tree.java. File akan menjalankan fungsi percabangan
yang akan digunakan dalam file Tree.java.
j) Tree.class
	File Tree.class merupakan file hasil compile Tree.java. File akan menjalankan fungsi yang sama
dengan file Tree.java.
k) Tree.java
	File Tree.java merupakan file untuk melakukan sortir menggunakan Binary Search Tree (BST) atau
pencarian pohon biner. Inti dari penggunaan BST adalah dengan menetapkan value root node pada value
pertama yang dimasukkan. Kemudian sortir akan berjalan berdasarkan lebih besar dan lebih kecil.
Terdapat 3 metode sortir data, diantaranya, Pre Order, In Order, dan Post Order.
	Metode Pre Order digunakan untuk melakukan sortir pada data setelah data diurutkan mulai dari
pemanggilan data pada node sebelah kiri BST. Metode In Order digunakan untuk melakukan sortir pada
data sesuai urutan dari yang terkecil. Metode Post Order digunakan untuk melakukan sortir pada data
diurutkan mulai dari node-node BST kemudian root node.
