3. Deskripsi File
a) Binary.class
	File Binary.class merupakan file hasil compile Binary.java. File akan menjalankan fungsi yang sama
dengan file Binary.java.
b) Binary.java
	File Binary.java merupakan file untuk melakukan sorting binary. Sorting binary membagi 2 dari
suatu list, sehingga pencarian akan mengerucut ke suatu sisi dari list atau array yang digunakan. File
menggunakan list angka sederhana dan beberapa key untuk mencoba. Selanjutnya, hasil akan ditampilkan
sesuai dengan hasil sorting.
c) BinarySearch.class
	File BinarySearch.class merupakan file hasil compile BinarySearch.java. File akan menjalankan
fungsi yang sama dengan file BinarySearch.java.
d) BinarySearch.java
	File BinarySearch.java merupakan file untuk melakukan sorting binary. File sama dengan fungsi
file Binary.java. User akan diminta untuk memasukkan banyak angka dan angka yang akan diurutkan.
Selanjutnya, hasil akan ditampilkan sesuai dengan hasil sorting.
e) SelectionSort.class
	File SelectionSort.class merupakan file hasil compile SelectionSort.java. File akan menjalankan
fungsi yang sama dengan file SelectionSort.java.
f) SelectionSort.java
	File SelectionSort.java merupakan file untuk melakukan selection sorting. File akan diperiksa satu
per satu mulai dari yang paling kiri. Iterasi dilakukan sebanyak data yang dimasukkan. User akan diminta
untuk memasukkan banyak angka dan angka yang akan diurutkan. Selanjutnya, hasil akan ditampilkan sesuai
dengan hasil sorting.
