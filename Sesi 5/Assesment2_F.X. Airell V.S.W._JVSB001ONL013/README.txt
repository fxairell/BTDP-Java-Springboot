3. Deskripsi File
a) Barang.class
	File Barang.class merupakan file hasil compile Barang.java. File akan menjalankan fungsi yang sama
dengan file Barang.java. File merupakan penunjang dari file PointofSales.java.
b) Barang.java
	File Barang.java merupakan file untuk implementasi encapsulation menggunakan getter. File diisi
dengan variabel atribut barang dagang dan getter dari atribut barang dagang.
c) Diskon.class
	File Diskon.class merupakan file hasil compile Diskon.java. File akan menjalankan fungsi yang sama
dengan file Diskon.java.
d) Diskon.java
	File Diskon.java merupakan file untuk melakukan perhitungan diskon sederhana. Diskon akan
diberikan pada harga minimal tertentu. User akan diminta untuk memasukkan total belanja. Selanjutnya,
total tagihan setelah mendapatkan diskon akan ditampilkan.
e) Kabisat.class
	File Kabisat.class merupakan file hasil compile Kabisat.java. File akan menjalankan fungsi yang
sama dengan file Kabisat.java.
f) Kabisat.java
	File Kabisat.java merupakan file untuk melakukan perhitungan tahun kabisat. User akan diminta
untuk memasukkan tahun yang ingin dicek. Selanjutnya, hasil akan ditampilkan apakah tahun kabisat atau
bukan. Informasi mengenai berapa tahun lagi untuk mencapai tahun kabisat akan ditampilkan apabila tahun
yang dimasukkan bukan merupakan tahun kabisat.
g) PointofSales.class
	File PointofSales.class merupakan file hasil compile PointofSales.java. File akan menjalankan
fungsi yang sama dengan file PointofSales.java.
h) PointofSales.java
	File PointofSales.java merupakan file untuk melakukan perhitungan belanja sederhana dalam bentuk
tabel. File disertai dengan deklarasi objek atribut barang. File akan menampilkan tabel barang dan
harga. User kemudian diminta untuk memasukkan jumlah barang yang akan dibeli, berikut dengan indeks dan
jumlah barangnya. File akan menampilkan hasil belanja dengan harga total yang harus dibayarkan. File
menggunakan encapsulation yang terhubung dengan file Barang.java dan TampilBarang.java.
i) TampilBarang.class
	File TampilBarang.class merupakan file hasil compile TampilBarang.java. File akan menjalankan
fungsi yang sama dengan file TampilBarang.java.
j) TampilBarang.java
	File TampilBarang.java merupakan file untuk implementasi encapsulation. File diisi fungsi
penampilan barang yang diambil dari class Barang.
k) UrutkanDuduk.class
	File UrutkanDuduk.class merupakan file hasil compile UrutkanDuduk.java. File akan menjalankan
fungsi yang sama dengan file UrutkanDuduk.java.
l) UrutkanDuduk.java
	File UrutkanDuduk.java merupakan file untuk melakukan pengurutan tempat duduk sederhana. File akan
menampilkan denah awal beserta keterangan tempat duduk. User kemudian diminta untuk memasukkan siapa
yang akan duduk pada posisi mana. File akan menampilkan denah baru dengan keterangan peserta yang duduk.
m) UrutkanTigaAngka.class
	File UrutkanTigaAngka.class merupakan file hasil compile UrutkanTigaAngka.java. File akan
menjalankan fungsi yang sama dengan file UrutkanTigaAngka.java.
n) UrutkanTigaAngka.java
	File UrutkanTigaAngka.java merupakan file untuk melakukan pengurutan angka terbesar, terkecil, dan
menghitung rata-rata dari tiga angka. User akan diminta untuk memasukkan 3 angka. File kemudian
menampilkan hasil perhitungan dan pengurutan angka tersebut.
