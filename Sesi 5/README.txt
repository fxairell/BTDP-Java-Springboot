3. Deskripsi File
a) AktivitasPagi.class
	File AktivitasPagi.class merupakan file hasil compile AktivitasPagi.java. File akan menjalankan
fungsi yang sama dengan file AktivitasPagi.java. File merupakan file untuk menunjang
AktivitasPagi.java.
b) AktivitasPagi.java
	File AktivitasPagi.java merupakan file pengenalan penggunaan interface untuk polymorphism. File
interface hanya ada method-method yang nantinya akan digunakan pada kelas lain.
c) AnakUmur1Tahun.class
	File AnakUmur1Tahun.class merupakan file hasil compile AnakUmur1Tahun.java. File akan menjalankan
fungsi yang sama dengan file AnakUmur1Tahun.java.
d) AnakUmur1Tahun.java
	File AnakUmur1Tahun.java merupakan file pengenalan penggunaan untuk polymorphism dengan
overriding. File memuat isi dari fungsi interface pada file AktivitasPagi.java.
e) Cetak.class
	File Cetak.class merupakan file hasil compile Cetak.java. File akan menjalankan fungsi yang sama
dengan file Cetak.java.
f) Cetak.java
	File Cetak.java merupakan file pengenalan penggunaan overriding method dengan menggunakan
pernyataan sederhana untuk memanggil class Cewek dan Cowok dan fungsi nyanyiLagu() yang terdapat pada
kedua class.
g) CetakAngka.class
	File CetakAngka.class merupakan file hasil compile CetakAngka.java. File akan menjalankan fungsi
yang sama dengan file CetakAngka.java.
h) CetakAngka.java
	File CetakAngka.java merupakan file pengenalan penggunaan overloading method dengan menggunakan
pernyataan sederhana untuk memanggil fungsi maxNumber() pada satu class yang sama dengan tujuan yang
berbeda.
i) Cewek.class
	File Cewek.class merupakan file hasil compile Cetak.java. File akan menjalankan fungsi
nyanyiLagu() yang terdapat pada file Cetak.java.
j) ClassIntro.class
	File ClassIntro.class merupakan file hasil compile ClassIntro.java. File akan menjalankan fungsi
yang sama dengan file ClassIntro.java.
k) ClassIntro.java
	File ClassIntro.java merupakan file pengenalan penggunaan objek dan class. Terdapat deklarasi
objek secara prosedural dan OOP. File juga disertai dengan beberapa contoh deklarasi class.
l) ClassIntro$makanan.class
	File ClassIntro$makanan.class merupakan file hasil compile ClassIntro.java. File hanya sebagai
contoh dan tidak dapat dijalankan.
m) ClassIntro$Nilai.class
	File ClassIntro$Nilai.class merupakan file hasil compile ClassIntro.java. File hanya sebagai
contoh dan tidak dapat dijalankan.
n) ClassIntro$people.class
	File ClassIntro$people.class merupakan file hasil compile ClassIntro.java. File hanya sebagai
contoh dan tidak dapat dijalankan.
o) Cowok.class
	File Cowok.class merupakan file hasil compile Cetak.java. File akan menjalankan fungsi
nyanyiLagu() yang terdapat pada file Cetak.java.
p) Driver.class
	File Driver.class merupakan file hasil compile Driver.java. File akan menjalankan fungsi yang sama
dengan file Driver.java.
q) Driver.java
	File Driver.java merupakan file pengenalan penggunaan polymorphism. File juga diserta dengan
contoh polymorphism, diantaranya class sendiri, class lain, dan interface.
r) Induk.class
	File Induk.class merupakan file hasil compile Induk.java. File akan menjalankan fungsi yang sama
dengan file Induk.java.
s) Induk.java
	File Induk.java merupakan file pengenalan penggunaan polymorphism. File merupakan file parent.
t) Latihan.class
	File Latihan.class merupakan file hasil compile Latihan.java. File akan menjalankan fungsi yang
sama dengan file Latihan.java.
u) Latihan.java
	File Latihan.java merupakan file latihan sekaligus pengenalan penggunaan inheritance. File
digunakan untuk melakukan perhitungan luas dan keliling dari beberapa bangun datar, seperti persegi,
persegi panjang, lingkaran, dan segitiga.
v) Lingkaran.class
	File Lingkaran.class merupakan file hasil compile Latihan.java. File merupakan penunjang untuk
file Latihan.java.
w) Lingkaran.java

x) Makanan.class

y) Makanan.java

z) Manusia.class
