3. Deskripsi File
a) DataConversion.class
	File DataConversion.class merupakan file hasil compile DataConversion.java. File akan menjalankan
fungsi yang sama dengan file DataConversion.java.
b) DataConversion.java
	File DataConversion.java merupakan file dengan deskripsi konversi data string menjadi integer.
Terdapat 4 cara dalam mengubah data string menjadi integer. Masing-masing cara dipisahkan oleh spasi
sebanyak 1. File tidak dapat dijalankan karena terdapat masalah dalam penggunaan fungsi Interger.
Sehingga, seluruh class masih dalam bentuk notes.
c) FormatString.class
	File FormatString.class merupakan file hasil compile FormatString.java. File akan menjalankan
fungsi yang sama dengan file FormatString.java.
d) FormatString.java
	File FormatString.java merupakan file untuk menggabungkan dua data string sederhana menjadi satu
keluaran string bersama. Terdapat 3 cara dalam menggabungkan data string.
e) Latihan_Java.class
	File Latihan_Java.class merupakan file hasil compile Latihan_Java.java. File akan menjalankan
fungsi yang sama dengan file Latihan_Java.java.
f) Latihan_Java.java
	File Latihan_Java.java merupakan file untuk latihan dalam mengelola list data dengan bahasa
pemrograman Java. Terdapat 4 latihan dalam mengelola list, diantaranya menambahkan, mengubah,
mengambil, menghapus, dan cek kondisi elemen pada list.
g) ListSetMap.class
	File ListSetMap.class merupakan file hasil compile ListSetMap.java. File akan menjalankan fungsi
yang sama dengan file ListSetMap.java.
h) ListSetMap.java
	File ListSetMap.java merupakan file untuk pengenalan pada berbagai tipe daftar data, diantaranya
List, Set, dan Map.
i) PrintVsPrintln.class
	File PrintVsPrintln.class merupakan file hasil compile PrintVsPrintln.java. File akan menjalankan
fungsi yang sama dengan file PrintVsPrintln.java.
j) PrintVsPrintln.java
	File PrintVsPrintln.java merupakan file untuk membandingkan fungsi Print dan Println.
k) program.class
	File program.class merupakan file hasil compile StructureMethod.java. File akan menjalankan fungsi
yang sama dengan file StructureMethod.java.
l) StatementsExpression.class
	File StatementsExpression.class merupakan file hasil compile StatementsExpression.java. File akan
menjalankan fungsi yang sama dengan file StatementsExpression.java.
m) StatementsExpression.java
	File StatementsExpression.java merupakan file untuk membandingkan Statements dan Expression.
Terdapat contoh Statements dan Expression sederhana.
n) StructureMethod.java
	File StructureMethod.java merupakan file untuk mengenalkan hierarki bahasa pemrograman. Terdapat
package, import, class dan public class void. Class diisi dengan Statement dan Expression sederhana
untuk mengisi class.
o) Switch.class
	File Switch.class merupakan file hasil compile Switch.java. File akan menjalankan fungsi yang sama
dengan file Switch.java.
p) Switch.java
	File Switch.java merupakan file untuk mengenalkan penggunaan switch pada java 8. Terdapat 2
optional case sederhana untuk menguji nilai, dan 1 nilai default.
q) SwitchNew.class
	File SwitchNew.class merupakan file hasil compile SwitchNew.java. File akan menjalankan fungsi
yang sama dengan file SwitchNew.java.
r) SwitchNew.java
	File SwitchNew.java merupakan file untuk mengenalkan penggunaan switch pada java 17. Terdapat 2
optional case sederhana untuk menguji nilai, dan 1 nilai default. Terdapat menggunakan mismatch agar
file tidak mengalami error saat ada pernyataan di luar case dan tidak jatuh ke default.
