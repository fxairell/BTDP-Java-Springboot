1. Definisi
a) File dengan ekstensi java merupakan file berisi coding dengan bahasa pemrograman Java.
b) File dengan ekstensi class merupakan file hasil compile file dengan ekstensi Java.
c) File dengan ekstensi xml merupakan file data dengan tag digunakan untuk menyimpan data.

2. Daftar File
- AddressBook.hbm.xml
- AddressBook.java
- App.java
- hibernate.cfg.xml

3. Deskripsi File
a) AddressBook.hbm.xml
	File AddressBook.hbm.xml merupakan file untuk melakukan ORM.
b) AddressBook.java
	File AddressBook.java merupakan file untuk mendefinisikan variabel data dengan ekstensi xml.
Terdapat setter dan getter untuk melakukan penarikan dan penampilan nilai.
c) App.java
	File App.java merupakan file untuk aplikasi utama untuk melakukan migrasi dengan hibernate.
d) hibernate.cfg.xml
	File hibernate.cfg.xml merupakan file untuk migrasi dengan hibernate.
