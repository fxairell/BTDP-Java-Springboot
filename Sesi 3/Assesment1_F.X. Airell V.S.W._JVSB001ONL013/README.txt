3. Deskripsi File
a) Aritmatika.class
	File Aritmatika.class merupakan file hasil compile Aritmatika.java. File akan menjalankan fungsi
yang sama dengan file Aritmatika.java.
b) Aritmatika.java
	File Aritmatika.java merupakan file untuk melakukan perhitungan artimatika sederhana sesuai dengan
tugas yang diberikan. Selanjutnya, hasil akan ditampilkan sebagaimana hasil dari perhitungan aritmatika.
c) IncrementDecrement.class
	File IncrementDecrement.class merupakan file hasil compile IncrementDecrement.java. File akan
menjalankan fungsi yang sama dengan file IncrementDecrement.java.
d) IncrementDecrement.java
	File IncrementDecrement.java merupakan file untuk melakukan penambahan dan pengurangan berkala
atau disebut dengan increment-decrement. Terdapat 4 kali proses untuk masing-masing increment-decrement.
Selanjutnya, hasil akan ditampilkan sebagaimana hasil dari penambahan atau pengurangan berkala.
e) LuasLingkaran.class
	File LuasLingkaran.class merupakan file hasil compile LuasLingkaran.java. File akan menjalankan
fungsi yang sama dengan file LuasLingkaran.java.
f) LuasLingkaran.java
	File LuasLingkaran.java merupakan file untuk melakukan perhitungan luas linkaran sederhana.
Seluruh variabel telah ditetapkan sebelumnya. Selanjutnya, hasil akan ditampilkan sebagaimana hasil dari
perhitungan luas lingkaran.
g) PenjumlahanXY.class
	File PenjumlahanXY.class merupakan file hasil compile PenjumlahanXY.java. File akan menjalankan
fungsi yang sama dengan file PenjumlahanXY.java.
h) PenjumlahanXY.java
	File PenjumlahanXY.java merupakan file untuk melakukan perhitungan aritmatik sederhana menggunakan
variabel X dan y. Nilai y1 dan y2 akan dimasukkan oleh user. Fungsi untuk X1 dan X2 telah ditetapkan
sebelumnya. Selanjutnya, hasil akan ditampilkan sebagaimana hasil dari perhitungan sederhana tersebut.
	Bagian kedua merupakan lanjutan dari perhitungan ini, untuk menentukan nilai boolean dari beberapa
ekspresi yang diberikan. Selanjutnya, hasil akan ditampilkan sebagaimana hasil dari perbandingan
tersebut.
i) PerbandinganTrueFalse.class
	File PerbandinganTrueFalse.class merupakan file hasil compile PerbandinganTrueFalse.java. File
akan menjalankan fungsi yang sama dengan file PerbandinganTrueFalse.java.
j) PerbandinganTrueFalse.java
	File PerbandinganTrueFalse.java merupakan file untuk melakukan perbandingan sederhana dari
beberapa pernyataan. Variabel dan pernyataan telah ditetapkan sebelumnya. Selanjutnya, hasil akan
ditampilkan sebagaimana hasil perbandingan sederhana.
