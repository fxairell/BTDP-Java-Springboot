3. Deskripsi File
a) BreakContinue.class
	File BreakContinue.class merupakan file hasil compile BreakContinue.java. File akan menjalankan
fungsi yang sama dengan file BreakContinue.java.
b) BreakContinue.java
	File BreakContinue.java merupakan file pengenalan penggunaan Break dan Continue dalam suatu loop.
File disertai dengan beberapa contoh penggunaan Break dan Continue menggunakan iterasi dengan
pernyataan sederhana.
c) Latihan.class
	File Latihan.class merupakan file hasil compile Latihan.java. File akan menjalankan fungsi yang
sama dengan file Latihan.java.
d) Latihan.java
	File Latihan.java merupakan file untuk menampilkan beberapa contoh latihan, diantaranya
menampilkan logo bintang, bilangan urut, bilangan ganjil urut, dan perulangan. Latihan merupakan
kesimpulan dari penggunaan berbagai fungsi perulangan for, while, dan do while.
e) NestedLoop.class
	File NestedLoop.class merupakan file hasil compile NestedLoop.java. File akan menjalankan fungsi
yang sama dengan file NestedLoop.java.
f) NestedLoop.java
	File NestedLoop.java merupakan file pengenalan fungsi nested loop atau loop dalam loop. Perulangan
menggunakan pernyataan sederhana sebagai contoh.
g) PerulanganForeach.class
	File PerulanganForeach.class merupakan file hasil compile PerulanganForeach.java. File akan
menjalankan fungsi yang sama dengan file PerulanganForeach.java.
h) PerulanganForeach.java
	File PerulanganForeach.java merupakan file pengenalan fungsi for each. Perulangan menggunakan
pernyataan sederhana sebagai contoh.
i) PerulanganWhile.class

j) PerulanganWhile.java
