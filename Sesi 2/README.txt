3. Deskripsi File
a) CekGraduation.class
	File CekGraduation.class merupakan file hasil compile CekGraduation.java. File akan menjalankan
fungsi yang sama dengan file CekGraduation.java.
b) CekGraduation.java
	File CekGraduation.java merupakan file untuk melakukan perhitungan kelulusan suatu ujian
sederhana. User akan diminta untuk memasukkan nama dan nilai. Proses perhitungan kelulusan ditentukan
dengan nilai dasar 78. Lulus akan dicapai apabila lebih dari 78, sementara tidak lulus sebaliknya.
Selanjutnya, hasil akan ditampilkan dengan nama yang diinputkan user beserta hasil kelulusannya.
c) ConditionalLogic.class
	File ConditionalLogic.class merupakan file hasil compile ConditionalLogic.java. File akan
menjalankan fungsi yang sama dengan file ConditionalLogic.java.
d) ConditionalLogic.java
	File ConditionalLogic.java merupakan file pengenalan logika kondisional if. File ini juga berisi
pengenalan mengenai fungsi logika dan variabel boolean dengan beberapa contoh disertai keterangannya.
e) Equals.class
	File Equals.class merupakan file hasil compile Equals.java. File akan menjalankan fungsi yang sama
dengan file Equals.java.
f) Equals.java
	File Equals.java merupakan file pengenalan fungsi sama dengan atau equals dengan beberapa metode.
Terdapat 3 metode penggunaan sama dengan. Selanjutnya, hasil akan ditampilkan dengan hasil penggunaan
sama dengan.
g) Hadiah.class
	File Hadiah.class merupakan file hasil compile Hadiah.java. File akan menjalankan fungsi yang sama
dengan file Hadiah.java.
h) Hadiah.java
	File Hadiah.java merupakan file untuk mengecek perhitungan pada suatu transaksi belanja. User akan
diminta untuk memasukkan total belanja. Program akan menghitung apakah pembeli mendapatkan hadiah dari
hasil belanjanya. Selanjutnya, hasil akan ditampilkan sebagai tanda apakah pembeli mendapatkan hadiah
atau ucapan terima kasih.
i) HitungGrade.class
	File HitungGrade.class merupakan file hasil compile HitungGrade.java. File akan menjalankan fungsi
yang sama dengan file HitungGrade.java.
j) HitungGrade.java
	File HitungGrade.java merupakan file untuk melakukan perhitungan grade dari suatu nilai. User akan
diminta untuk memasukkan nilai yang akan dilakukan perhitungan grade. Program akan menghitung grade
berdasarkan ketentuan nilai untuk grade. Selanjutnya, hasil akan ditampilkan sebagai grade untuk nilai.
k) IfOptional.class
	File IfOptional.class merupakan file hasil compile IfOptional.java. File akan menjalankan fungsi
yang sama dengan file IfOptional.java.
l) IfOptional.java
	File IfOptional.java merupakan file pengenalan fungsi kondisional bercabang atau if else. Kondisi
yang diberikan adalah pernyataan sederhana. Selanjutnya, hasil akan ditampilkan sebagai keluaran
pernyataan sederhana.
m) Kasir.class
	File Kasir.class merupakan file hasil compile Kasir.java. File akan menjalankan fungsi yang sama
dengan file Kasir.java.
n) Kasir.java
	File Kasir.java merupakan file untuk melakukan perhitungan diskon pada suatu transaksi belanja.
User akan diminta untuk memasukkan apakah pembeli memiliki kartu member atau tidak, dan total belanja.
Program akan menghitung berapa banyak diskon yang diberikan sesuai perhitungan diskon yang ditentukan.
Selanjutnya, hasil akan ditampilkan sebagai total pembayaran beserta perhitungan diskon.
