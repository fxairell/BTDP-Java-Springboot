3. Deskripsi File
a) JavaReadXmlSaxEx.java
	File JavaReadXmlSaxEx.java merupakan file untuk melakukan parsing data dan membaca data dengan
ekstensi xml pada folder DemoSAX. File digunakan sebagai file utama dengan metode SAX.
b) MyHandler.java
	File MyHandler.java merupakan file untuk membaca data dengan ekstensi xml.
c) MyRunner.java
	File MyRunner.java merupakan file untuk melakukan parsing utama data dengan ekstensi xml. Seluruh
source code parsing terdapat pada file ini.
d) User.java
	File User.java merupakan file untuk mendefinisikan variabel data dengan ekstensi xml. Terdapat
setter dan getter untuk melakukan penarikan dan penampilan nilai.
e) users.xml
	File users.xml merupakan file data yang menyimpan data user.
f) Employee.java
	File Employee.java merupakan file untuk mendefinisikan variabel data dengan ekstensi xml. Terdapat
setter dan getter untuk melakukan penarikan dan penampilan nilai.
g) ParseKnownXMLStructure.java
	File ParseKnownXMLStructure.java merupakan file untuk melakukan parsing data dan membaca data
dengan ekstensi xml pada folder DemoDOM. File digunakan sebagai file utama dengan metode DOM.
h) employees.xml
	File employees.xml merupakan file data yang menyimpan data user.
i) JavaReadXmlChallange.java
	File JavaReadXmlChallange.java merupakan file untuk melakukan parsing data dan membaca data dengan
ekstensi xml pada folder DemoChallange. File digunakan sebagai file utama dengan metode SAX.
j) ReadXmlAlexaApi.java
	File ReadXmlAlexaApi.java merupakan file untuk melakukan parsing data dan membaca data pada sebuah
website bernama Alexa pada folder DemoAlexa. File digunakan sebagai file utama dengan metode DOM.
