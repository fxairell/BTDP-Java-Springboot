1. Definisi
a) File dengan ekstensi java merupakan file berisi coding dengan bahasa pemrograman Java.
b) File dengan ekstensi class merupakan file hasil compile file dengan ekstensi Java.
c) File dengan ekstensi xml merupakan file data dengan tag digunakan untuk menyimpan data.
d) File dengan ekstensi html merupakan file data dengan tag digunakan untuk menampilkan data.
e) File dengan ekstensi css merupakan file data dengan tag digunakan untuk merubah penampilan data.
f) File dengan ekstensi sql merupakan file data query untuk manipulasi database.
g) File dengan ekstensi properties merupakan file bawaan untuk pembentukan aplikasi yang dapat
   mengatur properti yang digunakan aplikasi untuk berkomunikasi.

3. Deskripsi File
a) BelajarSpringBootstrapApplication.java
	File BelajarSpringBootstrapApplication.java merupakan file untuk aplikasi utama untuk menggunakan
springboot dalam menampilkan beberapa pesan sederhana dalam tampilan HTML.
b) WelcomeController.java
	File WelcomeController.java merupakan file controller menghubungkan pesan tampilan dengan view.
View dalam modul ini terdapat pada file HTML.
c) main.css
	File main.css merupakan file untuk merubah penampilan data HTML.
d) welcome.xml
	File welcome.xml merupakan file untuk menampilkan pesan sederhana.
e)  BelajarSpringBootstrapApplicationTest.java
	File BelajarSpringBootstrapApplicationTest.java merupakan file untuk melakukan testing aplikasi
utama saat dijalankan perintah clean, install, dan test.
f) pom.xml
	File pom.xml merupakan file profile Maven yang digunakan sebagai properti pada Maven. File ini
menampilkan berbagai versi, properti, dan dependensi dari Maven dan aplikasi yang digunakan.
g) MahasiswaController.java
	File MahasiswaController.java merupakan file controller menghubungkan pesan tampilan dengan view.
View dalam modul ini terdapat pada file HTML.
h) MahasiswaDao.java
	File MahasiswaDao.java merupakan file untuk merapikan dan menyusun tampilan data.
i) Mahasiswa.java
	File Mahasiswa.java merupakan file model untuk menyatakan mana variabel apa. Model digunakan
agar dapat terhubung dengan controller dan dapat diolah pada database.
j) LatihanSpringBootstrapApplication.java
	File LatihanSpringBootstrapApplication.java merupakan file untuk aplikasi utama untuk menggunakan
springboot dalam menampilkan tabel sederhana dalam tampilan HTML. Tabel dapat ditambah, diubah, dan
dihapus datanya sesuai keinginan user.
k) V1.0_201908061400_init_tables.sql
	File V1.0_201908061400_init_tables.sql merupakan file untuk menginisialisasi tabel. Tabel dibuat
dan disesuaikan dengan tampilan yang diinginkan.
l) V1.1_201908051500_init_tables.sql
	File V1.1_201908051500_init_tables.sql merupakan file untuk membuat tabel data sederhana. Data
pada tabel nantinya akan ditampilkan pada web service.
m) detail_form.html
	File detail_form.html merupakan file untuk menampilkan detail tabel data sederhana.
n) form.html
	File form.html merupakan file untuk menampilkan form pengisian data tabel sederhana.
o) index.html
	File index.html merupakan file untuk menampilkan tabel data sederhana.
p) application.properties
	File application.properties merupakan file penjelas dan karakteristik dari aplikasi. Properti
aplikasi dijelaskan dalam hal berhubungan atau bertukar data dengan service lainnya.
q) LatihanSpringBootstrapApplicationTest.java
	File LatihanSpringBootstrapApplicationTest.java merupakan file untuk melakukan testing aplikasi
utama saat dijalankan perintah clean, install, dan test.
