2. Daftar File

3. Deskripsi File
a) NumberGenerator.java
	File NumberGenerator.java merupakan file untuk melakukan generasi angka acak. Angka akan
dihasilkan dari rentang tertentu. File digunakan untuk menunjang file ThreadNumber.java dan
ThreadNumberTest.java.
b) ThreadHello.java
	File ThreadHello.java merupakan file pengenalan multithreading asynchronous. Hello akan dipanggil
sebagai hasil dari pemanggilan fungsi.
c) ThreadNumber.java
	File ThreadNumber.java merupakan file pengenalan multithreading synchronous. Angka acak sebanyak 3
angka akan dipanggil sebagai hasil dari pemanggilan fungsi.
d) ThreadWorld.java
	File ThreadWorld.java merupakan file pengenalan multithreading asynchronous. World akan dipanggil
sebagai hasil dari pemanggilan fungsi.
e) ThreadHelloTest.java
	File ThreadHelloTest.java merupakan file untuk testing file ThreadHello.java. Fungsi akan
dijalankan sebanyak 5 kali sebagai testing multithreading.
f) ThreadNumberTest.java
	File ThreadNumberTest.java merupakan file untuk testing file ThreadHello.java. Fungsi akan
dijalankan sebanyak 3 kali sebagai testing multithreading.
g) ThreadWorldTest.java
	File ThreadWorldTest.java merupakan file untuk testing file ThreadHello.java. Fungsi akan
dijalankan sebanyak 5 kali sebagai testing multithreading.
